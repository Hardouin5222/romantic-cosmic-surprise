# Güncelleme Rehberi

Bu dosyaları GitHub'a yükleyince Vercel otomatik olarak siteyi yeniden yayınlar.

## GitHub üzerinden güncelleme

1. GitHub'da `romantic-cosmic-surprise` reposuna gir.
2. Değiştirilecek dosyaya tıkla.
3. Sağ üstte kalem simgesine bas.
4. Yeni kodu yapıştır.
5. En alttaki **Commit changes** butonuna bas.
6. Vercel otomatik deploy başlatır.

## Klasör veya yeni dosya ekleme

1. GitHub repo ana sayfasında **Add file > Upload files** seç.
2. Dosya veya klasörleri sürükle bırak.
3. **Commit changes** de.

## Müzik ekleme

Müzik dosyasının adı kesinlikle şu olmalı:

```text
public/music.mp3
```

GitHub'a public klasörünün içine `music.mp3` olarak yükle. Sonra Vercel otomatik günceller.

## Vercel kontrol

Vercel > Project > Deployments kısmında en üstte yeşil `Ready` yazınca yeni sürüm yayındadır.
