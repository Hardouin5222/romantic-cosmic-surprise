# Güncelleme

1. GitHub'da eski `src` klasörünü sil veya içindeki dosyaların tamamını bu paketteki `src` ile değiştir.
2. Ana dizindeki `package.json` dosyasını bu pakettekiyle değiştir.
3. `index.html` dosyasını bu pakettekiyle değiştir.
4. `public/og-cover.png` dosyasını yükle.
5. İstersen `public/music.mp3` ekle.
6. Commit changes.
7. Vercel otomatik günceller.
8. Siteyi cache kırarak aç: https://romantic-cosmic-surprise.vercel.app/?v=3
