# Romantic Cosmic Surprise V3

Derin gece mavisi + altın romantik tek sayfa web deneyimi.

## Yayınlama

GitHub'daki mevcut repoda eski dosyaların üzerine bu paketteki dosyaları yükle.
Özellikle `src` klasörünü tamamen yenisiyle değiştir.

Vercel otomatik deploy eder.

## Müzik

`public/music.mp3` dosyasını kendi telifsiz/izinli müziğinle ekle. Dosya yoksa site çalışır, sadece müzik çalmaz.
